\# CodeAlpha Python Programming Internship



This repository contains the projects completed during my Python Programming Internship at CodeAlpha. Each project focuses on solving real-world problems while strengthening fundamental and intermediate Python programming skills, including automation, file handling, data processing, and graphical user interface development.



\## 🚀 Projects



\### 1. Hangman Game



A classic text-based word guessing game developed using Python. The application features randomized word selection, attempt tracking, input validation, and dynamic word updates, providing an engaging way to demonstrate core programming concepts such as loops, conditionals, and string manipulation.



\### 2. Stock Portfolio Tracker



A portfolio management application that allows users to track stock investments and calculate total portfolio value. The project utilizes dictionaries, user input handling, arithmetic operations, and CSV file generation to provide a practical introduction to financial data management.



\### 3. Email Extractor



An automation tool that uses Regular Expressions (Regex) to identify and extract email addresses from text files. The application demonstrates efficient text processing, pattern matching, and file handling techniques while automatically saving extracted results to an output file.



\### 4. CodeAlpha Virtual Assistant



A professional GUI-based desktop chatbot developed using Python and Tkinter. The application features a modern user interface, intent-based response handling, fuzzy string matching for typo tolerance, chat statistics, and dynamic user interaction, showcasing object-oriented programming and event-driven application development.



\## 🛠️ Technologies Used



\* \*\*Python\*\* – Core programming language used for application development.

\* \*\*Tkinter\*\* – Desktop GUI development and event handling.

\* \*\*Regular Expressions (Regex)\*\* – Pattern matching and text extraction.
